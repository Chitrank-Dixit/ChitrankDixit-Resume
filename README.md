## Files
This repository contains:
* *.pdf*: PDF copy of my Résumé.
* *.tex*: LaTeX source of my Résumé (needs *resume.cls* in the same directory).
* *resume.cls*: LaTeX style file (to accompany *.tex*).
* *README*: This readme file.
